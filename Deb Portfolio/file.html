Closer Look at my Apps

Mapper

My mapper map is an app that has the ability to map out any area, city, and will show business addresses. Specifically when the app loads; there is a search bar in which you would place a zip code or keyword you would want to look up. Then a tableview is generated with matching options. Click on the tableview cell of the correct option and the map shows you exactly where it is.

Notes

My Notes app is like the basic note app. The user has the ability to add new notes with date displayed, edit existing notes, save and delete notes. This note app also has the ability to save pictures.

ToDo LIST

My ToDo List app was similar to the Notes app; however with the todo list app, you could do the following:

1. Add, delete, edit, and save tasks

2. Place the tasks in categories (home, work, and other)

3. Mark a task complete-set a reminder and choose a due date

4. Add a picture to a todo task

5. Hide completed tasks

TIC TAC TOE

This is the basic Tic Tac Toe Game. This App will tell the user if it is X's or O's turn

The app will also keep score and allow the user to change the color of the X's and O's

Secret Santa

The Secret Santa App was my first Mac OS app. This app was my first team collaboration app.

Our team wanted an App that users could use anonymously to post a need.

In the app the user has the ability to add a need, edit, save, and delete a need.

This app lists the needs in a tableview for other users to see, read, and commit to meet.

Also, this app allows users to donate money. All information is stored on Parse because Firebase was not compatible with MacOS
